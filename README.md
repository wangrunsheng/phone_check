# phone_check

A Flutter Application to test mobile phones for recycling purpose.

## Fuctions

The UI is copied from 爱易收.

Some fuctions can be tested in this App:

- [ ] RAM size
- [ ] Vibration
- [ ] Micophone
- [ ] Speaker
- [ ] GPS
- [ ] Compass
- [ ] WIFI
- [ ] Acceleration Sensor
- [ ] Gyroscope
- [ ] Flash
- [ ] Front Camera
- [ ] Rear Camera
- [ ] Telephone
- [ ] Charging
- [ ] Touching
- [ ] Display
